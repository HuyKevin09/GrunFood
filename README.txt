GRUNMEAL 

Etapes à suivre  : 

* Installer Node.js
* Installer Visual Studio ou WebStorm
* Télécharger l'application Expo sur un téléphone
* Connecter le PC et le téléphone au même réseau wifi
* Télécharger le dossier du projet
* Lancer npm-install --legacy-peer-deps dans Visual Studio
* Lancer npm-start
* Scanner le QR code grâce au téléphone
* Utiliser l'application
